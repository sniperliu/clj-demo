# clj-demo

A Clojure project uses to demo clojure development workflow with Emacs and Docker.

## Requirement

Docker/Boot2Docker/Docker Composer installed
Emacs and Cider installed

## Usage

```bash
$ boot2docker up
$ eval "$(boot2docker shellinit)"
$ docker-composer up
```

Emacs
M-x cider-connect
